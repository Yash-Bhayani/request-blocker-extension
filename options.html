💡 Note on Manifest V3 Limitations:
This extension is built using the modern Manifest V3 architecture and the Declarative Net Request API for maximum privacy and performance. Because of how Manifest V3 handles network-level blocking, rules can only be applied to root domains. You can separate multiple root domains using commas.
